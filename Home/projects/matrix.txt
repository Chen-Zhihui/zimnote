====== matrix ======
Created 星期四 23 八月 2018

线性变换， 线性变换在基A下的表示，表示为矩阵  T[X] = [X]A
基[X] = [x1, x2, x3, ..., xn] 
在基[Y]上的表示 T[Y] = [Y] B
基底变换 [Y] = [X] C， C为过度矩阵
坐标变换 新坐标 y = inverse(C) x

同一个变换在不同基底下的表示矩阵相似 B = inverse(C) A C
线性变换在不同基下的表示矩阵不同，但这些矩阵相似, A~B

定理1.17 任意矩阵与上三角矩阵相似
定理1.24 变换T在某一基下是对角阵的充要条件是T有n个线性无关的特征向量。T的矩阵表示相似于对角阵

欧氏空间，度量->夹角->正交。
度量矩阵，基底的内积组成的矩阵，在基底[X]下，两个向量表示为a, b, (a,b）= transpose(a) A b, A为度量矩阵
如果基底取为标准正交积，则度量矩阵为单位阵
Schmid正交化方法：由任意基底构造标准正交积。相当的过度矩阵是一个上三角阵
正交变换：变换前后两个向量内积相同。在标准正交基上的矩阵表示是正交矩阵
正交矩阵：transport(Q)=inverse(Q)
特征值为1，特征向量正交
标准正交积：（ei, ej) = delt(i,j)
正交矩阵与单位矩阵相似，正交变换即以原点为中心的旋转变换

对称变换 （Tx, y) = (x, Ty)
对称矩阵
对称变换在标准正交基下的表示是对称矩阵
对称矩阵具有不同的特征值，和正交的特征向量
对称变换与对角矩阵相似

酉空间，线性空间，复数域上的线性空间，定义的内积
酉变换，相当于正交变换
酉对称变换，相当于对称变换
定理1.41 任意矩阵相似于上三角阵
正规矩阵 H(Q) Q = Q H(Q)
